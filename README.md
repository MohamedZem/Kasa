# 📖 À propos

**Kasa** est le cinquième projet réalisé dans le cadre de la formation **Développeur Web OpenClassrooms**.

L'objectif était de développer une **Single Page Application (SPA)** de location immobilière en utilisant **React**, **React Router** et **Sass**.

Le projet met l'accent sur la création de composants réutilisables, la navigation côté client et l'organisation d'une application React moderne.

---

# 🎯 Objectifs

- Développer une application avec React.
- Mettre en place une navigation avec React Router.
- Créer des composants réutilisables.
- Organiser une architecture React modulaire.
- Utiliser Sass pour structurer les styles.
- Respecter les bonnes pratiques de développement.

---

# ✨ Fonctionnalités

- 🏠 Affichage dynamique des logements
- 📄 Consultation des fiches détaillées
- 🖼️ Carrousel d'images
- ⭐ Affichage des notes des logements
- 🏷️ Gestion des tags
- 📂 Menus déroulants (Collapse)
- 🚫 Gestion des routes inexistantes (404)

# ⚙️ Installation

### 1. Cloner le dépôt

```bash
git clone https://github.com/MohamedZem/Kasa.git
```

### 2. Accéder au projet

```bash
cd Kasa
```

### 3. Installer les dépendances

```bash
npm install
```

### 4. Lancer le serveur de développement

```bash
npm run dev
```

L'application sera accessible à l'adresse suivante :

```text
http://localhost:5173
```
